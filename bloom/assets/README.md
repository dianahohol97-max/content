# bloom focus — Shorts assets

## music-calm.mp3
Place a royalty-free calm background track here named exactly `music-calm.mp3`.
It will be mixed UNDER the voiceover at 12% volume and looped to fit each Short.

Sources for free music (download as MP3, rename to music-calm.mp3):
- YouTube Audio Library (Studio → Audio Library) — filter "Calm" / "Ambient"
- Pixabay Music (pixabay.com/music) — search "calm lofi" or "soft piano"
- Uppbeat (uppbeat.io) — free with attribution

If no music file is present, Shorts are built with voiceover only (still works).
